## ⚡ QUICK START (5 MINUTES)

```bash
# 1. Navigate to project
cd D:\Personal\TradingBot

# 2. Run database migrations (creates new tables)
dotnet ef database update -p TradingBot.Persistence -s TradingBot

# 3. Configure API keys (not stored in code!)
dotnet user-secrets init --project TradingBot
dotnet user-secrets set "Binance:ApiKey" "your-key" --project TradingBot
dotnet user-secrets set "Binance:ApiSecret" "your-secret" --project TradingBot

# 4. Start the app
dotnet run --project TradingBot

# 5. Test in another terminal
curl http://localhost:5000/api/system/status
```

## 🎓 UNDERSTANDING THE ARCHITECTURE

### Layer Responsibilities

```
WEB API LAYER (Controllers)
    ↓
APPLICATION LAYER (Services, Business Logic)
    ↓
DOMAIN LAYER (Entities, Interfaces)
    ↓
INFRASTRUCTURE LAYER (Binance, External APIs)
    ↓
PERSISTENCE LAYER (Database)
```

### Data Flow Example

```
User Request
    ↓
TradeController.OpenTrade()
    ↓
RiskManagementService.CanTradeToday()
    (Check daily loss limit, position size, etc.)
    ↓
BinanceTradeExecutionService.OpenTradeAsync()
    (Execute trade on Binance testnet)
    ↓
Save Trade & Order to Database
    ↓
TradeMonitoringWorker (every 10 sec)
    (Check if SL/TP hit)
    ↓
Auto-close if needed
```
